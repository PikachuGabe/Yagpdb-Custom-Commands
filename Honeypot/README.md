# How to use:
- Copy the text of https://github.com/PikachuGabe/Yagpdb-Custom-Commands/blob/main/Honeypot/honeypot.yagcc into a new custom command.
- Restrict where the script runs using YAGPDB's channel restrictions. 
- Set the custom command to use Regex with a pattern of `.*`.
- Add any role or user IDs you want to be ignored to the bypass variables.
- Set the `configured` variable to true.
